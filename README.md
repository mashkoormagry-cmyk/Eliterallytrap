# Eliterallytrap Website

Multi-page React + Tailwind site for the YouTube channel **Eliterallytrap**.

## Features
- Home page with featured/latest video
- All Videos page (YouTube playlist embed)
- Star Promoter code page (shows code `Eliterallytrap`)
- About and Contact pages
- Responsive, Vercel-ready

## Deploy to Vercel (free)
1. Create a GitHub repo and push this project.
2. Sign up at https://vercel.com and "Import Project" from GitHub.
3. Choose the repo, set Framework to "Vite", build command `npm run build` and output directory `dist`.
4. Click Deploy. Your site will be live at `https://<your-project>.vercel.app` (you can rename the project to `eliterallytrap` to get `eliterallytrap.vercel.app` if available).
